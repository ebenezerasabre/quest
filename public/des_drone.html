Radio transmitter

Description

In most of my projects I need wireless control so I decided to build this multifunctional radio controller which can be used for pretty much everything

It features:

  • NRF24L01 +PA +LNA
  • Arduino Pro Mini
  • IMU MPU6050
  • 2x Potentiometers
  • 2x Joysticks
  • 6x Push Buttons
  • 2x Toggle Switches
  • 2x Potentiometers
