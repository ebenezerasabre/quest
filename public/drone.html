Drone

Description

Drones have always facinated me since I was a child, so I decided to build one. I ordered the frame from an online store since I didn't have a 3D printer to print my own. The speed controller is made of Arduino nano and MPU6050 mudule. The radio receiver on the other hand consist of an Arduino nano and NRF24L01 transreceiver module.

It features:

  • Plastic frame
  • Radio receiver
  • Speed controller
  • Lithium ion battery

Receiver module

I decided to use a radio receiver I made from a previous project. The NRF24L01 transreceiver module receives incoming signal and pass it as input to the Arduino nano.

Speed Controller

Instead of using a pre-made speed controller that came with the frame, I decided to build one using an arduino nano and an MPU6050 module which is made of a gyroscope and accelerometer on the same silicon die.
